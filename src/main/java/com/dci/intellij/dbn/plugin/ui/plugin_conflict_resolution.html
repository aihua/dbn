PLUGIN CONFLICT
  • Database Navigator - third party
  • Database Tools and SQL - JetBrains standard distribution
Both Database Navigator (DBN) and the Database Tools and SQL plugins, provide very similar database functionality. Having both plugins installed, can lead to ambiguous and at times conflicting user experience. You are advised to disable one of them.

Database Navigator is NOT a complementary product to (nor is ist integrated with) the standard distribution database support. It offers only few additional features and a slightly different user experience, while being less integrated with the other IDE features and language support.
